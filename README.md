# Docker-cgwire

Docker containers for [Kitsu](https://kitsu.cg-wire.com/) and [Zou](https://zou.cg-wire.com/).

### Usage

#### First launch

The **-i/--init** flag init Zou and the database

```bash
bash build.sh -i
```

#### Standard launch

```bash
bash build.sh
```

Default credentials:

* login: admin@example.com
* password: mysecretpassword
